# OpenCV Learning Projects

A collection of OpenCV and Computer Vision programs. These projects demonstrate fundamental image processing and object detection techniques using Python and OpenCV.



## 📌 About

This repository contains beginner to intermediate Computer Vision programs created to learn how OpenCV detects and processes various facial features and objects using Haar Cascade classifiers and image processing techniques.

The purpose of this repository is to strengthen my understanding of:
- Image Processing
- Computer Vision Fundamentals
- Object Detection
- Real-time Webcam Processing
- OpenCV Library


## 🛠️ Technologies Used

- Python 3.12
- OpenCV
- NumPy
- Haar Cascade Classifiers


## 📂 Programs Included

### 👤 Face Detection
Detects human faces in images using Haar Cascade classifiers.

### 👁️ Eye Detection
Identifies eyes from facial images.

### 👀 Face & Eye Detection
Simultaneously detects faces and eyes in an image.

### 🎥 Live Face & Eye Detection
Performs real-time face and eye detection using a webcam.

### 👃 Nose Detection
Detects the nose region from facial images.

### 👄 Mouth Detection
Detects the mouth region using OpenCV.

### 😊 Multi-Facial Feature Detection
Detects face, eyes, nose, and mouth together in a single image.

### 🔍 Pupil Detection
Locates pupils from eye images using image processing techniques.

### 🚦 Stop Sign Detection
Detects traffic stop signs from road images.


## 🎯 Learning Objectives

These projects helped me understand:

- Reading and processing images using OpenCV
- Face detection using Haar Cascades
- Eye, nose, mouth, and pupil detection
- Real-time webcam video processing
- Drawing bounding boxes around detected objects
- Working with grayscale images
- Image preprocessing techniques
